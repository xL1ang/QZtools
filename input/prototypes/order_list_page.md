# 订单列表页参考原型

## 页面布局

顶部：筛选栏（状态下拉框 + 日期范围选择 + 搜索框 + 查询按钮）
中部：数据表格
底部：分页组件

## 表格字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 订单号 | 文本 | 可点击跳转详情 |
| 客户名 | 文本 | - |
| 商品数量 | 数字 | 汇总件数 |
| 订单金额 | 金额 | 保留2位小数 |
| 状态 | 标签 | 不同状态不同颜色 |
| 创建时间 | 日期 | yyyy-MM-dd HH:mm |
| 操作 | 按钮组 | 查看/取消 |

## 交互说明

- 点击"查看"跳转订单详情页
- 点击"取消"弹出确认对话框
- 表格支持按列排序
