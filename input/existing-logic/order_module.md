# 订单管理模块

## 功能概述

订单管理模块负责处理客户下单到发货的全流程。

## 现有功能

### 1. 订单创建
- 客户选择商品、填写收货地址
- 系统校验库存，生成订单号
- 订单状态初始为"待付款"

### 2. 订单列表
- 支持按状态筛选（待付款/待发货/已发货/已完成）
- 支持按时间范围查询
- 显示字段：订单号、客户名、金额、状态、创建时间

### 3. 订单状态流转
- 待付款 → 待发货（客户支付成功）
- 待发货 → 已发货（仓库确认发货）
- 已发货 → 已完成（客户确认收货）
- 任意状态 → 已取消（客户主动取消或超时未付款）

### 4. 订单详情
- 基本信息：订单号、客户、金额、状态
- 商品明细：商品名、数量、单价、小计
- 物流信息：快递公司、运单号、物流轨迹
